Over de jaren heb ik verschillende tools ontwikkeld. De meeste heb ik ondertussen verwijderd omdar er door derden betere tools beschikbaar zijn gesteld. Wat er overblijft heb hier onder beschreven inclusief de openbare links naar github.


Zabbix template voor Dell switches

Een zabbix template welke zelf alle optische interfaces door middel van een geautomatiseerde discovery kan vinden en vervolgens de demping in dBm de spanning in Volt en de stroom in mA kan loggen en monitoren. De toepassing is handig als je onverklaarbare storingen op een glasvezel netwerk hebt en wil vallideren het optische pad in orde is.


Zabbix template voor generieke UPS

De standaard APC UPS templates binnen zabbix voldeden niet aan mijn behoeftes dus heb ik zelf een template gebouwd. Helaas is deze template gebouwd zonder discovery het is dus onbekend of het ook goed werkt op andere merken.


Database error na zabbix upgrade fix

Na een upgrade van zabbix kreeg ik verschillende database foutmeldingen. Na wat zoeken heb ik een simpele methode gevonden om de database weer werkend te krijgen.


Curl naar influxDB met python

Dit is een heel simpel voorbeeld hoe je in python gegevens met curl ophaalt en ze vervolgens in influxDB zet. Dit is voor mij meestal de basis voor het starten van een interactie met een RestfullAPI


Verloop dagen van een SSL certificaat

Niet heel bijzonder, maar een check binnen hoeveel dagen een ssl certificaat verloopt met behulp van openSSL, handig voor in een cronjob


Een script dat een escape of backslash voor een special teken plaatst

Soms moet ik voor het aanpassen van scripts een escape character toevoegen. Dit script is een basis om een search en add te doen.
